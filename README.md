# AstrBot 话痨王插件 (Talkative King)

一个用于 AstrBot 的群组发言统计插件，能够统计群员每日发言次数并生成精美的排行榜图片。

## ✨ 功能特点

- **精美图片榜单**：生成包含头像、昵称、排名和发言次数的排行榜图片，支持自定义背景。
- **完美支持特殊符号**：内置 `pilmoji` 支持，完美渲染 Emoji 和颜文字（如 `( 。v_v 。)`），告别乱码。
- **智能触发**：
  - **模糊匹配**：忽略指令中的空格、标点符号（如 `今日 壁画王！` 也能触发）。
  - **抗干扰**：无论是否 @机器人，或者指令包含前缀，都能精准响应。
- **精准时间控制**：强制使用 **北京时间 (UTC+8)**，每日 **00:00** 准时重置数据并归档。
- **数据安全**：
  - **原子化保存**：防止断电或崩溃导致数据丢失。
  - **智能昵称保护**：自动识别并保留用户的有效昵称，防止被“QQ号”覆盖。
- **昨日回溯**：支持查看昨日榜单，精准还原昨日数据。

## 🚀 安装方法

1. 将 `astrbot_plugin_talkative_king` 文件夹放入 AstrBot 的 `data/plugins/` 目录下。
2. 确保安装了必要的依赖库：
   ```bash
   pip install -r requirements.txt
   ```
   *(主要依赖：Pillow, pilmoji, emoji)*
3. 重启 AstrBot 或在 WebUI 管理面板重载插件。

## 📖 使用指南

### 📅 今日排行榜
查看当天的发言统计排行。

- **触发关键词**（支持任意组合，如 `/今日发言` 或 `@Bot 今日壁画王`）：
  - `今日壁画王`
  - `今日发言`
  - `今日排行`
  - `今日发言排行榜`

### 🗓️ 昨日排行榜
查看昨天的发言统计排行。

- **触发关键词**：
  - `昨日壁画王`
  - `昨日发言`
  - `昨日排行`
  - `昨日发言排行榜`

## ⚙️ 高级配置

- **背景图片**：你可以替换 `assets/background.jpg` 来自定义榜单背景，或者在代码中指定自定义背景路径。
- **字体支持**：默认使用 `msyh.ttc` (微软雅黑)，如遇生僻字可自动回退到系统字体。
